About Me

Perkenalkan nama saya Husni Darmawan, nama panggilan saya Wawan. Saya masih duduk dibangku SMK YPK kelas XII. Teman-teman, untuk mengimbangi lajunya era dunia digital saat ini, jangan sampai kita ketinggalan informasi-informasi terupdate dari sosial media lainnya. Maka dari itu saya mengambil jurusan Tehnik Komputer dan Jaringan. Setidaknya saya paham tentang perangkat dasar komputer dan komponen didalamnya, lalu bisa mempelajari software dan aplikasi lainnya dengan mudah. Itulah alasan saya mengambil jurusan ini.

Semakin berkembangnya dunia digital, semakin mudah kita mendapatkan informasi. Saya ingin membantu teman-teman yang ingin belajar web coding menggunakan HTML dan CSS. Untuk itu saya harap teman-teman mendukung artikel saya ini dengan cara gabung ke grup whatsapp yang telah saya sediakan linknya. Dan juga bisa teman-teman download beberapa jenis aplikasi untuk pembuatan web, yang berada di menu Products. Terimakasih untuk kesempatan dan perhatian teman-teman yang telah berkunjung ke web ini. Salam semangat pantang menyerah.
